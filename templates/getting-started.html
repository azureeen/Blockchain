Getting Started with the Flask Blockchain Project

This guide will walk you through setting up and running the Flask Blockchain project on your local machine using Docker.

Prerequisites

  • Docker
  • Docker Compose

These tools are necessary to create and manage the containers in which the Flask application runs.

Installation Instructions

  1. Unzip the Project Folder: First, unzip the downloaded project folder. This folder contains all the necessary files, including the `docker-compose.yml` file which is crucial for setting up the Docker environment.
  2. Navigate to the Project Directory: Open your command line or terminal and navigate to the directory containing the unzipped project files. Use the `cd` command to change directories. For example: 
    cd path/to/your/project-directory
  3. Starting the Application with Docker Compose: Once you are in the project directory, start the application by running the following Docker Compose command: 
    docker-compose up -d

Using the Application

With the Docker containers running, the Flask app should now be up and accessible. You can interact with the application by visiting the various links provided in the project documentation.

Exploring the Project

Try out the different functionalities of the Flask Blockchain project:

  • Add transactions
  • View the blockchain
  • Check specific transactions
  • View the last 10 transactions

Each feature is accessible through its respective link or endpoint in the Flask application.

Stopping the Application

When you're done exploring the application, you can stop the Docker containers by running:

docker-compose down

This command shuts down the containers and cleans up any resources used.

Troubleshooting

If you encounter any issues while setting up or running the project, ensure that:

  • Docker is running on your system.
  • You are in the correct project directory in your command line or terminal.
  • The ports required by the Flask application are not being used by another service.
